Historical Match Agent
ACTIVE
Performs hybrid search (semantic + keyword) to find similar past requirements

Search Strategy

  • Semantic Search (70%): Vector similarity using embeddings
  • Keyword Search (30%): BM25-based keyword matching
  • Searches across Epics, Estimations, and TDDs collections
  • Returns top matches with score breakdown

Vector Database

Database: ChromaDB Embedding Model: all-minilm (Ollama) Collections: epics, estimations, tdds Dimensions: 384 (MiniLM)
